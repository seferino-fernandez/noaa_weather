# NOAA Weather client and CLI

## Development

### Running Tests

Run tests:

```sh
cargo test
```

### Linting and Formatting

Run Clippy:

```sh
cargo clippy
```

Run Rustfmt:

```sh
cargo fmt
```

## Usage

View help for the CLI:

```sh
cargo run -- --help
```

## Documentation

View documentation for the CLI:

```sh
cargo run -- --help
```
